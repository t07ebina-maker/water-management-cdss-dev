# CLAUDE.md

このファイルは、リポジトリ内で作業する Claude Code (claude.ai/code) へのガイダンスを提供します。

## 概要

Claude Code の作業用汎用フォルダです。初期状態ではプロジェクトのコードベースは存在しません。必要に応じてファイルやプロジェクトがここに作成されます。

## 環境

- プラットフォーム: Windows 11（Bash では Unix シェル構文を使用 — スラッシュは `/`、`/dev/null` など）
- シェル: bash

---

## デザイン規則

pptx・HTML・その他 UI を生成・編集する際は、必ず `./DESIGN.md` を最初に読み込み、
そこに定義された規則のみを使用すること。以下はその要点である。

### 禁止事項（即時確認）

- `./DESIGN.md` を読まずに色・フォント・レイアウトを独自に決定しないこと
- フッター保護ゾーン（y > 17.75cm）へのコンテンツ配置禁止
- スライドマスター（slideMaster1.xml）のフッター要素を改変・削除しないこと
- グラデーション背景・赤・紫系カラーの使用禁止
- MS ゴシック・MS 明朝への変更禁止

### アセットの場所

ロゴ・富士山・院章などのブランド画像は `./assets/` に格納されている。
pptx を新規作成する場合は `./assets/` からこれらを参照すること。

| ファイル        | 内容                         |
|-----------------|------------------------------|
| image2.jpeg     | 静岡市立静岡病院ロゴ（横型） |
| image3.emf      | 富士山シルエット             |
| image4.png      | 病院外観写真（小）           |
| image5.emf      | 院章マーク（小）             |
| image6.jpeg     | 院章マーク（大・表紙用）     |

### 作業完了チェックリスト

pptx 生成後:
  [ ] フッター 4 要素（外観写真・院章・ロゴ横・富士山）が全スライドに表示されている
  [ ] y > 17.75cm にコンテンツが侵食していない
  [ ] 游ゴシック／Calibri Light が使用されている
  [ ] カラーが #0068B3・#6FBA2C の 2 トーン構成である

HTML 生成後:
  [ ] ヘッダーが #0068B3 背景・白テキスト・緑ボーダー
  [ ] 本文フォントが游ゴシック系
  [ ] グラデーション・box-shadow が使用されていない
  [ ] ボタンが #0068B3 または #6FBA2C

## index.html 保護ルール

既存の `index.html` を上書き・削除しないこと。
変更が必要な場合は必ず `index_v{N}.html` としてバージョン番号を付与した
新ファイルを作成し、元ファイルを保持すること。
